#EscapeHellsKitchen

Name of game: Escape Hell's Kitchen

Group members: Ana Morais, Jasmine Rodelas, Samantha Georges, Malaika Joiner

Game Play:
- Our game has an escape room style, where the player is trapped in a kitchen
- Throughout the game play, the player will find notes written by an ex employee about his journey of escaping but with a twist at the end
- Some effects we want to implement are darkening sides of viewpoint, proximity sounds, footsteps, and jumpscares
- While working at the restaurant, the player has side tasks of making food for the boss
